## Overview

This repository contains the code for Race Car Dashboard Prototype v2, designed to provide immersive visualization and advanced telemetry analysis. This prototype was part of a larger project exploring how dashboard design impacts driver performance in racing simulations.

## Key Features

*   **Advanced Telemetry Visualizations:** Incorporates graphs and charts for deeper insights into vehicle performance.
*   **Immersive Design:** Aims to provide more comprehensive data without sacrificing usability.
*   **HTML, CSS, and JavaScript Implementation.**

## Technologies Used

*   HTML
*   CSS
*   JavaScript
